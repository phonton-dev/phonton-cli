# Phonton CLI v0.4.2

Phonton CLI 0.4.2 adds saved interactive sessions and a safer exit path for the local-first ADE loop.

## Changes

- [add] `phonton -r` / `phonton --resume` restores the latest saved TUI session for the current workspace.
- [add] `Ctrl+C` and top-level `Esc` now open an exit confirmation before Phonton ends the session.
- [add] Confirmed TUI exits save a per-workspace session snapshot with visible goals, ask state, Flight Log data, and token totals.
- [add] After exit, Phonton prints a session receipt with goals, completed/reviewing/failed counts, actual tokens used, estimated naive baseline tokens, estimated saved tokens, and best observed savings percentage.
- [ui] Restore the normal ANSI Shadow Phonton splash logo and show the CLI version as a muted TUI label.
