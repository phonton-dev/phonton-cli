# Phonton CLI v0.4.0

Phonton CLI 0.4.0 is the Accountability Handoff Alpha. It adds product
surfaces for understanding short prompts, showing what changed, and keeping
durable evidence for completed tasks.

## Changes

- [new] Mention workspace files directly in the TUI prompt with `@src/lib.rs`,
  `@"notes file.md"`, or `@[design screenshot.png]`.
- [new] Attach small text files and images to a goal so workers receive the
  relevant context without pasting it manually.
- [new] Send image attachments through native vision payloads for providers
  that support them.
- [new] Generate a goal contract before work starts, including assumptions,
  acceptance criteria, likely files, and a quality floor.
- [new] Produce a review-ready handoff packet when work finishes, covering the
  result, changed files, verification, token usage, and known gaps.
- [new] Show the handoff in the TUI center panel instead of forcing users to
  inspect the Flight Log to understand the outcome.
- [new] Persist outcome ledgers in the local task store and include them in
  history and review output.
- [fix] Restore historical v0.3.1 release notes so the public release history
  remains intact.

## Install

```bash
npm install -g phonton-cli@0.4.0
```

or:

```bash
cargo install --git https://github.com/phonton-dev/phonton-cli --tag v0.4.0 phonton-cli --locked --force
```
