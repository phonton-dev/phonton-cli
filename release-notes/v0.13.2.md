# Phonton CLI v0.13.2

v0.13.2 fixes the remaining chess benchmark stale-hunk path for existing
Vite/React repositories.

## Fixed

- Chess prompts that say to use the existing project stack now detect Vite and
  React from the workspace, even when the prompt does not literally say
  Vite/React/TypeScript.
- Existing Vite/React chess runs now use the compact Vite chess acceptance
  slices instead of the generic chess slices.
- App UI slices carry `Artifact: src/App.tsx`, so the first worker prompt
  includes the current `src/App.tsx` snapshot before asking the provider for a
  diff. This prevents stale first-line hunks like expecting
  `import { useState } from 'react'` when the actual file starts with
  `import './App.css'`.
