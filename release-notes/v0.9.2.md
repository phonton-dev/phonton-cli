# Phonton CLI v0.9.2

v0.9.2 is a hotfix for task-specific quality gates during goal execution.

## Fixed

- Quality-gate failures now trigger one targeted worker repair pass instead of
  immediately failing the task after normal verification passes.
- The repair pass receives the exact quality-gate failures as retry context.
- A verified chess implementation that misses reset/new-game behavior now gets
  a chance to repair that requirement before Phonton reports failure.

- `cargo test --locked --workspace`
