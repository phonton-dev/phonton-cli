# Phonton CLI v0.4.8

TUI polish release for memory labels, splash rendering, and exit summaries.

## Fixed

- Worker rows hide the internal `# Prior context from memory` preamble and display the actual subtask label.
- The PHONTON splash keeps the same ASCII art and gradient palette while using a stable logo phase to avoid gradient-animation artifacts.

## Changed

- The exit confirmation modal now includes a session summary with goal counts, token totals, estimated savings, and `phonton -r` resume behavior.

## Verification

- `cargo fmt --all -- --check`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo test --locked --workspace`
- `cargo build --release --locked -p phonton-cli`
