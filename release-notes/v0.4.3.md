# Phonton CLI v0.4.3

Phonton CLI 0.4.3 is a terminal-safety hotfix for the v0.4.2 session release.

## Changes

- [fix] Disable fastembed/Hugging Face download progress output during TUI goal runs, preventing `model.onnx` progress bars from writing over the input area.
- [fix] Use an ASCII-safe splash wordmark and spinner so Windows terminal rendering stays stable under Ratatui redraws.
- [fix] Route semantic-index setup failures through tracing instead of direct stderr writes while the TUI owns the terminal.
