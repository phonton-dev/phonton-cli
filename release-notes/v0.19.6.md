# Phonton CLI v0.19.6 - Interactive Clarification Questionnaire

v0.19.6 introduces a highly aesthetic, guided **Interactive Clarification Questionnaire** inside the terminal user interface (TUI). When goal specifications are vague (low confidence or unanswered questions), execution suspends, and the user is guided step-by-step to provide outstanding details directly in the console.

## Key Improvements

### 📝 TUI Interactive Clarification Mode
- **Guided Q&A Loop**: Low-confidence goals (confidence < 70% or unanswered requirements questions) no longer simply fail; they now display a prominent amber warning card.
- **Dynamic Entry**: Pressing `'C'` instantly shifts the active keyboard focus into a sleek clarification step questionnaire (`Mode::Clarify`).
- **Interactive Editing**: Supports standard character insertion, backspaces, cursor movements, and paste buffers for entering clarification answers.
- **Self-Refining prompts**: Completing the questionnaire automatically appends all Q&A details into the original goal description and programmatically triggers a clean planning rerun.
- **Robust State Cleanup**: Resetting/rerunning a refined goal clears previous statuses, flight logs, and checkpoint states to prevent any state leaks.

### 🧪 Verified Verification & Quality
- Added full unit test coverage validating keyboard event routing, buffer editing, answer stacking, and goal re-queue intent generations.
- Verified clean of all lints and 100% tests passing across all library and binary crates.
