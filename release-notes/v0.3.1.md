# Phonton CLI v0.3.1

Phonton CLI 0.3.1 is a visual polish patch for the terminal UI introduced in
the 0.3.x line.

## Changes

- The input bar no longer uses the native terminal cursor while Phonton is
  running. The TUI now draws a steady caret directly in the input row so the
  typing indicator does not blink.
- The splash logo uses a tighter ANSI-shadow PHONTON mark with stronger
  letter shapes and a subtle pixel-shadow row.
- The logo gradient animation is smoother: it ticks more often, moves more
  slowly, uses eased color transitions, and applies a softer shimmer pass.

## Install

```bash
npm install -g phonton-cli@0.3.1
```

or:

```bash
cargo install --git https://github.com/phonton-dev/phonton-cli --tag v0.3.1 phonton-cli --locked --force
```
