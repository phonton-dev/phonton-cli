# Phonton CLI v0.15.1

v0.15.1 is a hotfix for v0.15.0 TUI regressions and the generated Vite chess
seed path.

## Fixed

- Existing Vite/React chess rules seeds now write a real Vitest suite in
  `src/chessRules.test.ts`, fixing `No test suite found in file` failures.
- The compact top `phonton` gradient animates while a goal is active again.
- The Receipt focus shortcut row no longer wraps `d diff` onto a stranded line
  at common terminal widths.
- Flight Log scrolling works from tail mode with `PgUp`, arrow keys, and mouse
  wheel input.
