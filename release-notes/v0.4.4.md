# Phonton CLI v0.4.4

Phonton CLI 0.4.4 restores the normal TUI visual identity while keeping the terminal-corruption fix from 0.4.3.

## Changes

- [ui] Restore the animated ANSI Shadow Phonton splash logo.
- [ui] Restore compact header glyphs, Braille spinner, and unicode token-savings bar.
- [fix] Keep fastembed/Hugging Face download progress disabled during TUI goal runs so `model.onnx` progress cannot write over the screen.

## Verification

- `cargo fmt --all -- --check`
- `cargo test --locked -p phonton-cli splash_logo_is_compact_and_shadowed`
- `cargo test --locked -p phonton-cli renders_shadow_logo_on_wide_splash`
- `cargo test --locked -p phonton-cli renders_version_on_compact_header`
- `cargo test --locked --workspace`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo build --release --locked -p phonton-cli`
