# Phonton CLI v0.13.4

v0.13.4 fixes the remaining existing Vite/React chess benchmark failure where
the first provider diff could still corrupt `index.html` or `package.json`.

## Fixed

- Existing Vite/React chess runs now use a dedicated source-first contract
  instead of the empty-workspace scaffold contract.
- The first slice targets `src/chessRules.ts` and `src/chessRules.test.ts`.
  It does not advertise `package.json` or `index.html` as editable artifacts.
- Existing-workspace rules slices no longer request `chess.js`, preventing the
  model from pulling dependency/package edits into the first diff.
- App slices still include both `src/App.tsx` and `src/App.css` as artifacts so
  workers patch against current source and styles.
