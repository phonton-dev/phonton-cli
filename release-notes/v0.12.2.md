# Phonton CLI v0.12.2

v0.12.2 fixes an early-slice verification regression in the playable chess
benchmark path.

## Fixed

- Vite/React chess scaffold slices now request a starter `src/chessRules.ts`
  rules boundary and `src/chessRules.test.ts` smoke test so Vitest has a test
  file before `npm test` is expected to pass.
- Node verification now waits to run Vitest/Jest-style file-discovery scripts
  until a `.test.*` or `.spec.*` file exists, while still running custom npm
  test scripts and running Vitest/Jest once generated tests are present.
- Added regression coverage for the exact failure mode where `npm test` failed
  during the scaffold slice because no test files had been generated yet.
