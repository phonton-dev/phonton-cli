# Phonton CLI v0.15.5

v0.15.5 is a TUI hotfix for prompt typing, visible input position, and Ask
answer readability.

## Fixed

- Bare prompt letters such as `f`, `d`, `p`, and `r` now type normally. The
  corresponding Active-panel shortcuts moved to `Alt+F`, `Alt+D`, `Alt+P`,
  and `Alt+R`.
- The prompt bar now renders a static caret rectangle inside the input area
  while the native terminal cursor remains hidden.
- Ask-mode answers now style common inline markdown for bold, italic, and
  inline code spans instead of showing raw formatting markers.

## Notes

- `0 tok` remains expected for deterministic local fast paths, such as the
  known Vite chess seed slices. It means Phonton applied a locally verified
  patch without calling a provider.
