# Phonton CLI v0.4.1

Phonton CLI 0.4.1 is a trust-surface patch for the v0.4 accountability release.

## Changes

- [fix] Expose `goal_contract` at the top level of `phonton plan --json` output while keeping the nested planner payload intact.
- [fix] Extend the npm wrapper smoke test to run `phonton plan --json --no-memory` and fail if the GoalContract surface is missing.
