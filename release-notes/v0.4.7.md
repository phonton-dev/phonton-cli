# Phonton CLI v0.4.7

Patch release for Cloudflare Settings persistence.

## Fixed

- Cloudflare Account ID entries are now copied into the saved provider config.
- Goal runs and Settings saves use the same Settings-to-config sync helper, so provider name, model, API key, Account ID, base URL, and budget fields stay consistent.

- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo test --locked --workspace`
- `cargo build --release --locked -p phonton-cli`
