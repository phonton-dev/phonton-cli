# Phonton CLI v0.15.2

v0.15.2 is a hotfix for the existing Vite/React chess UI path and TUI cursor
rendering.

## Fixed

- Existing Vite/React chess UI slices now use a local verified app-shell seed
  for `src/App.tsx`, `src/App.css`, and `src/vite-env.d.ts` instead of spending
  a provider call on fragile generated CSS hunks.
- Repeated UI slices become zero-token no-ops after the local app shell is
  already current, so Phonton does not report noisy unchanged file rewrites.
- The TUI no longer requests a native terminal cursor during frame renders,
  removing the blinking prompt/header bar while preserving the compact
  `phonton` gradient animation.
