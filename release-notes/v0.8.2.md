# Phonton CLI v0.8.2

v0.8.2 is a prompt artifact and review-scroll usability patch.

## Added

- The Active receipt/code surface now scrolls with the mouse wheel,
  `PgUp` / `PgDn`, and `Home` / `End`, making large generated diffs readable
  without leaving the TUI.
- Pasted or dropped image file paths now collapse into image artifact chips such
  as `[image: board.png]` and are included in the submitted prompt as image
  artifacts.

## Changed

- Prompt artifact chips now use stable accent colors instead of plain white
  text in the prompt bar, goal sidebar, and Active header.

## Validation

- `cargo fmt --all -- --check`
- `cargo test --locked --workspace`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo build --release --locked -p phonton-cli`
- `target/release/phonton.exe version`
- `target/release/phonton.exe --help`
- `npm run test:npm-wrapper`
- `npm pack --dry-run`
