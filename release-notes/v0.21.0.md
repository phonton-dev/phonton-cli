# phonton-cli v0.21.0

## Desktop serve RPC

New JSON-RPC methods for Phonton Desktop:

- `tasks.list`, `tasks.get` — task history from SQLite store
- `workspace.info` — cwd, trust, config/store paths
- `config.get`, `config.save`, `config.path` — read/write `~/.phonton/config.toml` (masked keys on read)
- `trust.list`, `trust.grant` — workspace trust
- `extensions.list`, `extensions.read`, `extensions.write`, `extensions.validate` — user + workspace `.phonton` files

## CORS

All `phonton serve` routes include CORS headers (from v0.20.2).
