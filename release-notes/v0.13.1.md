# Phonton CLI v0.13.1

v0.13.1 is a generated-app token guard hotfix for the chess benchmark path.

## Fixed

- Explicit Vite + TypeScript + React chess prompts now keep the compact Vite
  chess acceptance-slice contract even when the workspace already contains a
  partial `package.json` or broken generated files from a previous attempt.
- Generated-app acceptance slices are classified as high-risk generated app
  work, so worker prompt budgets stay on the generated-app route instead of
  drifting into broader feature budgets.
- First-attempt generated web-app syntax failures in TSX, JSX, HTML,
  JavaScript, or TypeScript now stop before an automatic provider repair call.
  This turns the previous multi-thousand-token failed repair path into an
  honest failure with verifier evidence and an explicit retry path.
