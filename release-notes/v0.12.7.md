# Phonton CLI v0.12.7

v0.12.7 is a diff/review quality-of-life release.

## Added

- `phonton diff [latest|<task-id>]` exports verified unified diffs from
  persisted review-ready task events.
- `phonton diff --stat`, `phonton diff --name-only`, and `phonton diff --json`
  provide compact surfaces for scripts, reviews, and quick file inspection.
- `/diff`, `/code`, and the `d` key open the TUI Code focus without cycling
  through other panels.

## Changed

- Code focus headers now show file counts and `+/-` line totals, so the diff
  view is scannable before reading individual hunks.
- CLI help and README docs now describe the diff command and shortcuts.
