# Phonton CLI v0.13.3

v0.13.3 completes the existing Vite/React chess benchmark artifact fix.

## Fixed

- The scaffold slice for compact Vite chess plans now advertises every file it
  may touch: `package.json`, `index.html`, `src/main.tsx`, `src/App.tsx`,
  `src/chessRules.ts`, and `src/chessRules.test.ts`.
- Because generated-app workers read artifact-marked files before the first
  provider call, the first prompt now includes the actual current App file even
  before the later App UI slices run.
- This closes the stale-hunk path where the provider guessed an old
  `src/App.tsx` import line and Phonton spent thousands of tokens before
  failing syntax verification.
