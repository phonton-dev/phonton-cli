# Phonton CLI v0.4.5

Phonton CLI 0.4.5 fixes a goal-run panic in provider configuration.

## Changes

- [fix] Prevent a TUI crash when a goal run starts with an incomplete provider configuration.
- [fix] Cloudflare without an Account ID or full Workers AI base URL now fails the goal with a setup message instead of panicking at `unknown provider config`.
- [fix] Worker dispatch now builds per-tier provider configs from a validated template, preserving Cloudflare/custom endpoints and configured models.
