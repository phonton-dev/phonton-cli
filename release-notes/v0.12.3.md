# Phonton CLI v0.12.3

v0.12.3 fixes stale generated-test hunks in the playable chess benchmark path.

## Fixed

- Vite/React chess rules slices now carry both `src/chessRules.ts` and
  `src/chessRules.test.ts` as current artifacts when the worker may need to keep
  the rules API and tests aligned.
- Repair attempts now include the exact current file named in verifier
  diagnostics, so a mismatch in `src/chessRules.test.ts` gets repaired against
  the real file instead of retrying from compact error text alone.
- Added regressions for the stale removed-line mismatch seen in TypeScript
  verification after the scaffold slice generated a starter test file.
