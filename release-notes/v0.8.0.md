# Phonton CLI v0.8.0

v0.8.0 adds the prompt artifact paste system.

## Highlights

- Multiline and long clipboard paste no longer submits accidental goals.
- Bracketed paste is enabled in the TUI, so pasted blocks arrive as one event.
- Paste artifacts render as compact chips, for example `[paste: 3 lines, 14 chars]`.
- `Ctrl+V` imports clipboard text through the same artifact path.
- Pending paste artifacts can be inspected or removed before pressing `Enter`.
- Planner and worker prompts carry typed prompt artifacts separately from file attachments.

## Fix

Before this release, pasting:

```text
do x
do y
do z
```

could be interpreted as three Enter presses and queue three goals. Phonton now
stores that paste as one prompt artifact and waits for one explicit `Enter`.
