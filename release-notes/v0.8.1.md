# Phonton CLI v0.8.1

v0.8.1 is a hotfix for Windows terminal paste behavior.

## Fixed

- Some terminals, including Windows/VS Code terminal configurations, can send a
  multiline paste as fast `Enter` key events instead of a bracketed paste event.
  Phonton now detects those plain-text bursts and collapses them into one paste
  artifact, so a copied benchmark prompt does not queue every line as a
  separate goal.
- The TUI input channel now has more headroom for large paste bursts.
