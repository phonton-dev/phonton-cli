# Phonton CLI v0.15.3

v0.15.3 is a hotfix for existing Vite/React chess projects that already have a
placeholder `src/App.test.*` file.

## Fixed

- Existing Vite/React chess UI seeds now replace stale App placeholder tests
  when `src/App.test.*` or `src/App.spec.*` already exists, so the zero-token
  local App shell does not fail verification against old heading expectations.
- The seeded App test uses React server rendering and plain Vitest assertions,
  avoiding Testing Library, jsdom, and custom matcher assumptions.

## Verification

- `cargo fmt --all -- --check`
- `cargo test --locked --workspace`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo build --release --locked -p phonton-cli`
- `target/release/phonton.exe version`
- `target/release/phonton.exe --help`
- `npm run test:npm-wrapper`
- `npm pack --dry-run`
- Real Vite/Vitest fixture with stale `src/App.test.tsx`: `npm test -- --run`
  and `npm run build`
