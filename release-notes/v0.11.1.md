# Phonton CLI v0.11.1 - Benchmark Hotfix

v0.11.1 is a small trust-loop hotfix for the chess benchmark path.

## Fixed

- Chess quality gates now recognize symbol/abbreviation piece maps such as
  `K/Q/R/B/N/P` and Unicode chess glyphs as valid piece evidence.
- Empty-workspace goals that explicitly ask for HTML/web chess now target a
  static `index.html` artifact instead of the Python terminal-game default.
- Broad chess quality repair no longer auto-runs after an already-expensive
  attempt crosses 8k provider tokens. Phonton fails honestly and asks for an
  explicit `/retry` using compact diagnostics.

## Notes

This release does not add benchmark superiority claims. It tightens the
benchmark path so failures are cheaper and false quality failures are less
likely.
