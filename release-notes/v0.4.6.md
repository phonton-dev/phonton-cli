# Phonton CLI v0.4.6

Patch release for Cloudflare provider diagnostics.

## Fixed

- Settings connection tests now surface the real Cloudflare setup requirement when the Account ID or Workers AI base URL is missing.
- Failed goals now show their failure reason in the Active panel, so provider setup issues stay visible after the run stops.

- `cargo fmt --all -- --check`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo test --locked --workspace`
- `cargo build --release --locked -p phonton-cli`
