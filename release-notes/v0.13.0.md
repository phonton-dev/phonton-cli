# Phonton CLI v0.13.0

v0.13.0 is a polish release for workspace-aware Ask, steadier TUI rendering,
and first-class verified diff review surfaces.

## Added

- TUI Ask and `phonton ask` now build bounded read-only workspace context by
  default: explicit `@file` mentions, current goal diagnostics, workspace
  facts, a compact file map, and lexical file excerpts.
- Ask responses are prompted to cite inspected workspace paths and admit when
  the selected context is insufficient.
- `phonton ask --no-workspace` keeps the old stateless prompt path, and
  `phonton ask --json` emits answer/context metadata for scripts.
- `phonton diff [latest|<task-id>]` exports verified unified diffs from
  persisted review-ready task events, with `--stat`, `--name-only`, and
  `--json` compact modes.
- `/diff`, `/code`, and the `d` key open the TUI Code focus without cycling
  through other panels.

## Changed

- The compact `phonton` header freezes after a goal exists, fixing the rapid
  flashing/shimmer seen while using the TUI. Running status indicators still
  animate.
- Ask mode shows a compact `ctx:` line with selected workspace files and
  estimated context tokens.
- Code focus headers now show file counts and `+/-` line totals.
