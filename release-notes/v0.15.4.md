# Phonton CLI v0.15.4

v0.15.4 is a hotfix for existing Vite/React chess goals that passed local
verification but failed Phonton's playable-chess quality gate because the UI
only referenced named pieces dynamically.

## Fixed

- Existing Vite/React chess App shell seeds now render an accessible named-piece
  legend for king, queen, rook, bishop, knight, and pawn.
- The seeded App test asserts named piece evidence so future zero-token local
  chess UI seeds cannot pass verification while remaining invisible to the
  diff-based quality gate.

## Verification

- `cargo fmt --all -- --check`
- `cargo test --locked --workspace`
- `cargo clippy --locked --workspace --all-targets -- -D warnings`
- `cargo build --release --locked -p phonton-cli`
- `target/release/phonton.exe version`
- `target/release/phonton.exe --help`
- `npm run test:npm-wrapper`
- `npm pack --dry-run`
- Real Vite/Vitest fixture with stale `src/App.test.tsx`: `npm test -- --run`
  and `npm run build`
