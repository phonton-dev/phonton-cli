# Phonton CLI v0.7.0 Trust Loop Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.7.0 as a first-run trust-loop release where users can see a contract, deterministic demo receipt, run commands, and exportable review evidence.

**Architecture:** Reuse the existing CLI subcommand dispatcher and typed `GoalContract` / `HandoffPacket` spine. Add focused CLI modules for the deterministic demo and latest-run command surface; keep cross-crate types unchanged unless a behavior crosses crate boundaries.

**Tech Stack:** Rust, Tokio, Ratatui CLI crate, `phonton-types`, `phonton-store`, `phonton-sandbox`, serde JSON/Markdown rendering.
