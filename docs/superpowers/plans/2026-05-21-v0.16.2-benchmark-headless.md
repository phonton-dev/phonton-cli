# v0.16.2 Benchmark Headless Reliability Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `phonton goal --prompt-file` a reproducible, provider-honest benchmark path that captures baseline test evidence, stronger prompt-mentioned context, bounded repair evidence, and claim-safe benchmark exports.

**Architecture:** Keep the release scoped to the existing accountability spine. The planner extracts prompt-mentioned files/APIs into `GoalContract`, the headless CLI records baseline test evidence before dispatch when required, the worker receives bounded baseline/failing evidence as repair context, and benchmark export continues to decide claim eligibility from typed events and execution mode.

**Tech Stack:** Rust workspace, Tokio, existing `phonton-types`, `phonton-planner`, `phonton-cli`, `phonton-worker`, `phonton-verify`, PowerShell benchmark scripts.

```rust
fn prompt_mentioned_paths(text: &str) -> Vec<PathBuf> {
    let mut paths = Vec::new();
    for token in text.split(|ch: char| ch.is_whitespace() || matches!(ch, ',' | ';' | ':' | ')' | '(')) {
        let cleaned = token.trim_matches(|ch| matches!(ch, '`' | '\'' | '"' | '[' | ']' | '.'));
        if cleaned.contains('/')
            && cleaned
                .rsplit_once('.')
                .is_some_and(|(_, ext)| matches!(ext, "js" | "jsx" | "ts" | "tsx" | "rs" | "py" | "go" | "java" | "html" | "css"))
        {
            let path = PathBuf::from(cleaned);
            if !paths.iter().any(|existing| existing == &path) {
                paths.push(path);
            }
        }
    }
    paths
}

fn prompt_public_api_constraints(text: &str) -> Vec<String> {
    let mut constraints = Vec::new();
    for segment in text.split('`').skip(1).step_by(2) {
        let value = segment.trim();
        if value.contains('(') && value.contains(')') && value.len() <= 80 {
            constraints.push(format!("Preserve public API `{value}`."));
        }
    }
    constraints
}
```

Call these from `Goal::contract()` before returning `GoalContract`, deduping with existing likely files and appending API constraints to `acceptance_criteria`.

#[derive(Debug, Clone, PartialEq, Eq)]
struct HeadlessBaselineCommand {
    label: String,
    command: Vec<String>,
}

fn headless_baseline_commands(prompt: &str) -> Vec<HeadlessBaselineCommand> {
    let lower = prompt.to_ascii_lowercase();
    if !(lower.contains("run the tests first") || lower.contains("run tests first")) {
        return Vec::new();
    }
    if lower.contains("npm test") {
        return vec![HeadlessBaselineCommand {
            label: "npm test".into(),
            command: vec!["npm".into(), "test".into()],
        }];
    }
    Vec::new()
}
```

fn compact_headless_evidence(label: &str, exit_code: i32, output: &str) -> String {
    const MAX_CHARS: usize = 1400;
    let mut compact: String = output.chars().take(MAX_CHARS).collect();
    if output.chars().count() > MAX_CHARS {
        compact.push_str("\n[truncated]");
    }
    format!("baseline command `{label}` exit {exit_code}\n{compact}")
}
```
